# tmdb-movie-data-analysis

Exploratory data analysis of the TMDb movie dataset using Python,
Pandas, Numpy and Matplotlib.

## Questions Explored

- How are movie ratings distributed in the dataset?
- Which movies generated the highest revenue?
- Is there relationship between movie budget and revenue?
- How many movies are released per year?

- ## Tools used

- -Python
- Pandas
- Numpy
- Matplotlib
- Jupyter Notebook
